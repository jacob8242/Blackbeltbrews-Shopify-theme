# How to Download and Install the Shopify Theme

## Download the Theme

The complete theme has been packaged as a ZIP file: **blackbeltbrews-shopify-theme.zip** (661 KB)

### Option 1: Download from GitHub
1. Go to the repository on GitHub
2. Click on the file `blackbeltbrews-shopify-theme.zip`
3. Click the "Download" button

### Option 2: Download from Pull Request
1. Go to the Pull Request on GitHub
2. Click on "Files changed"
3. Find `blackbeltbrews-shopify-theme.zip` in the file list
4. Click on the file and download it

## Install in Shopify

Once you have downloaded the ZIP file:

1. **Go to your Shopify Admin**
   - Navigate to: Online Store > Themes

2. **Upload the Theme**
   - Click "Add theme" button
   - Select "Upload ZIP file"
   - Choose the `blackbeltbrews-shopify-theme.zip` file
   - Click "Upload"

3. **Activate the Theme** (optional)
   - Once uploaded, you can preview the theme
   - Click "Publish" to make it your live theme
   - Or click "Customize" to edit it first

## What's Included in the ZIP

The ZIP file contains all necessary Shopify theme files:

- **assets/** (88 files) - CSS, JavaScript, and SVG icons

Total: 239 files organized in proper Shopify 2.0 structure

## After Installation

After installing the theme, you may want to:

1. **Configure Theme Settings**
   - Go to Online Store > Themes > Customize
   - Adjust colors, fonts, and layout options

2. **Enable Shop Features**
   - Ensure Shop Pay is enabled in your Shopify Payments settings
   - Enable "Follow on Shop" in the footer section settings
   - Configure which payment methods to display

3. **Add Your Content**
   - Add products with images
   - Create collections
   - Customize homepage sections
